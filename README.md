# Schedule_Nurse3_Gallery
Sample Projects Gallery for Schedule Nurse‡V, free Windows Solver for Nurse Scheduling Problem.

# How to run the project

1. Install <a href= "https://www.nurse-scheduling-software.com/download/"> Schedule Nurse‡V</a> (You need Windows 10, 64 bit version.)

2. Click  "Open Project file from Github". 
<div align="center"><img src="https://raw.githubusercontent.com/sugawara-system/Schedule_Nurse3_Gallery/main/docs/images/github_viewer.png" width="1200"/></div>

3. Select your favorite project.
<div align="center"><img src="https://raw.githubusercontent.com/sugawara-system/Schedule_Nurse3_Gallery/main/docs/images/select_file_on_github.png" width="1200"/></div>

4. Before you solve, you should specify the file name to store the project. 
<div align="center"><img src="https://raw.githubusercontent.com/sugawara-system/Schedule_Nurse3_Gallery/main/docs/images/downloaded_file.png" width="1200"/></div>

5. Click "Solve"
<div align="center"><img src="https://raw.githubusercontent.com/sugawara-system/Schedule_Nurse3_Gallery/main/docs/images/solve_window.png" width="1200"/></div>
